# discography ( LEVEL 0 project)
#### [링크](https://badahertz52.github.io/discography/main)를 열어서 프로젝트를 확인하세요. 
## <LEVEL 0 프로젝트 소개>
  LEVEL 0 프로젝트는 예비 개발자로 공부한 언어, 기능을 개인적인 작업으로 구현해 보는 프로젝트로 해당 프로젝트 중 하나인 discography는  html. css, 바닐라 자바스크립트와 부분적인 react 사용을 통해  음악 작업물을 소개하는 홈페이지를 만드는 것을 목표로 하였다. 

## <홈페이지 소개>
  방탄소년단(BTS) 멤버들이 sound cloud에 공개하는 개인 작업물(협업제외)들을 소개하는 홈페이지이다. </br>
  홈페이지는 크게 메인 홈페이지와 멤버 개별 작업물 홈페이지로 구성되었다.</br>
  메인 홈페이지는 멤버마다의 작업물을  한 눈에 볼 수 있으며 믹스테이프(앨범 형식의 개인 작업물) 와 개별곡을 아이콘을 다르게 하여 구별할 수 있게 하였다.</br>
  개별 작업물 홈페이지는 개발시의 효율성을 생각하여 개별 작업물은 템플릿을 만들어 적용하였다.</br>
  개별 작업물 홈페이지에서 사용자는 해당 작업물을 보고 들을 수 있는 music 부분(믹스테이프인 경우 해당 믹스테이프 명으로, 개별곡일 경우  song이라는 명으로 되어 있다.) , 뮤직비디오와 관련 콘텐츠의 존재 여부에 따라 music video부분, behind부분 과 more 부분으로 구성되어  있다. 개인 작업물 중 곡만 공개되고 관련 비디오나 콘텐츠가 없는 경우 해당 곡들을 모아놓은 0.page가 존재하며 여러 멤버가 같이 작업한 곡의  경우는 more.page에서 소개하며 그 중에 "네시" 와 "땡"은 별도의 소개 페이지를 만들었다. 
</br>
## <사용한 기능>
#### 1. react를 이용한 헤더 재사용
  각 페이지의 상단부분, 헤더에는 다른 페이지로 이동할 수 있는 네이게이션 기능을 담당하고 있고, 해당 기능은 동일한 내용으로 페이지마다 반복적으로 사용된다. </br>
  따라서 반복적으로 사용되는 헤더부분을 관리하기 쉽고 코드의 반복을 피하기 위해 리액트를 활용하여 하나의 컴포넌트를 만들어 재사용할 수 있도록 하였다. </br>
  페이지의 전체에 리액트가 필요한 것이 아니므로 부분적으로 사용할 수 있는 방법을 사용하여 리액트를 사용하였다. </br>


#### 2. toggle 기능 - doDropdown, showContent, doPopup 함수 
  버튼을 눌렀을 때 관련 내용이 보이도록 하는 기능이 필요했지만 각 상황이 달라서 상활에 따라 맞는 함수가 필요했고 각 함수들이 여러 곳에서 재사용될 수 있도록 함수들을 만들었다.</br>  doDropwdown은 버튼을 클릭하며 바로 다음 형제 요소를 보여주는 기능이며, showContent는 여러 버튼들와 여러 내용들이 존재하는 경우 특정 버튼을 누르면 그에 맞는 내용을 보여주는 기능을 하며 이는 버튼들과 내용들을 담고 있는 각 부모 노트들을 형제요소로 하고 해당 버튼이 부모 노트에서 몇 번째인지 알아내어 내용들을 담고 있는 부모 노드에서 해당 인덱스를 가진 노트를 찾아내어 버튼에 맞는 내용이 보이도록 함으로써 작동한다. doPopup의 경우 doDropdown과 비슷하지만 doDropdown은 주로 네이게이션에서 사용되지만 doPopup은 팝업창처럼 작동하며 해당 내용창을 닫을 수 있는 Close 함수가 같이 작동된다. 

#### 3. slide 기능
  여러 컨텐츠들을 묶어서 보여주기 위해 slide 기능을 하는 함수를 만들었다.
