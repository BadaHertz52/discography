## <프로젝트 소개>
'LEVEL 0 프로젝트는 예비 개발자로 공부한 언어, 기능을 개인적인 작업으로 구현해 보는 프로젝트로 해당 프로젝트 중 하나인
discography는 html. css, 바닐라 자바스크립트만을 이용하여 음악 작업물을 소개하는 홈페이지를 만드는 것을 목표로 하였다. 

## <홈페이지 소개>
방탄소년단(BTS) 멤버들이 sound cloud에 공개하는 개인 작업물(협업제외)들을 소개하는 홈페이지이다. 
홈페이지는 크게 메인 홈페이지와 멤버 개별 작업물 홈페이지로 구성되었다.
메인 홈페이지는 멤버마다의 작업물을  한 눈에 볼 수 있으며 믹스테이프(앨범 형식의 개인 작업물) 와 개별곡을 아이콘을 다르게 하여 구별할 수 있게 하였다. 
개별 작업물 홈페이지는 개발시의 효율성을 생각하여 개별 작업물은 믹스테이프템플릿과 개별 곡만 있는 작업물용 템플릿을 만들어 적용하였다.
개별 작업물 홈페이지에서 사용자는 각 곡의 가사와 뮤직비디오 해당 곡과 관련된 라이브 영상등을 볼 수 있으며 상위에 음악 플랫폼에서 직접 음악을 들을 수 있도록 해놓았다.
